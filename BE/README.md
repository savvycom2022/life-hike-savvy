# Book Store Backend Application
## Description
This application is for book store business, which helps people to manage book, orders of book and payments
Application use NestJS framework & Stripe Library

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
